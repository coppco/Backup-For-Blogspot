# 工欲善其事必先利其器
现在iOS开发,大部分都是使用Xcode工具开发了,这里就是记录一下,从事iOS几年来的一些Xcode快捷键和Mac一些命令.

# <font color=orange>Xcode</font>
<!--more-->
* __<font size=5>选中Xocde, <font color=red>command + shift + N</font>  :    在已经打开的工程中另外新建一个工程</font>__

* __<font size=5>选中Xocde, <font color=red>command + N</font>   :   在已经打开的工程中新建一个文件file</font>__

* __<font size=5>选中Xocde, <font color=red>command + shift + 0</font>   :   打开苹果帮助文档</font>__

* __<font size=5>选中Xocde, <font color=red>command + shift + O</font>   :   搜索整个工程文件和接口文档</font>__

* __<font size=5>选中Xocde, <font color=red>command + S</font>   :   快速保存</font>__

* __<font size=5>选中Xocde, <font color=red>command + ,(逗号)</font>   :   打开Xcode偏好设置</font>__

* __<font size=5>选中Xocde, <font color=red>command + shift + K</font>   :   清理Xcode缓存等</font>__

* __<font size=5>选中Xocde, <font color=red>command + shift + J</font>   :   查看当前所在文件file</font>__

* __<font size=5>选中Xocde, <font color=red>control + 6</font>   :   查看当前方法,还可以输入方法搜索和跳转到方法等</font>__

* __<font size=5>选中Xocde, <font color=red>command + shift + F</font>   :   搜索当前工程</font>__

* __<font size=5>选中Xocde, <font color=red>command + F</font>   :   搜索当前文件file</font>__

* __<font size=5>选中Xocde, <font color=red>command + R</font>   :   编译并运行工程</font>__

* __<font size=5>选中Xocde, <font color=red>command + B</font>   :   编译工程</font>__

* __<font size=5>选中Xocde, <font color=red>command + .</font>   :   停止运行模拟器</font>__

* __<font size=5>OC开发的时候, 使用<font color=red>control + command + up、down(方向键)</font>   :   在.h和.m文件切换</font>__

* __<font size=5>选中一个file, <font color=red>command + option + return</font>   :    打开辅助编辑器, 例如在storyboard中使用
    就可以打开对应的file文件,然后就可以拖线了</font>__

* __<font size=5>选中一个file, <font color=red>按住option键,再选择另外一个file</font>   :    打开辅助编辑器</font>__

* __<font size=5>选中一个file, <font color=red>command + return</font>   :    关闭辅助编辑器</font>__

* __<font size=5>输入代码的时候使用<font color=red>tab</font>键  :   快速切换到下一个输入块</font>__

* __<font size=5>输入代码的时候使用<font color=red>esc</font>键  :   使用自动补全功能</font>__

* __<font size=5>选中代码, <font color=red>command + /</font>  :   使用注释功能</font>__

* __<font size=5>选中代码, <font color=red>control + I</font>  :   使代码格式化对齐</font>__

* __<font size=5>选中代码, <font color=red>command + [ 或 ]</font>  :   使代码左移/右移</font>__

* __<font size=5>选中代码, <font color=red>command + shift + [ 或 ]</font>  :   使代码上移/下移</font>__

* __<font size=5>光标移动到}, <font color=red>command + option + left、right(方向键)</font>  :   快速打开、叠起代码,也可以点击代码左侧位置</font>__

* __<font size=5>在xib或者storyboard选中控件, <font color=red>command + option + =</font>  :   更新控件约束</font>__

* __<font size=5>在xib或者storyboard输入的文本实现换行, <font color=red>option + enter</font>  :   xib或者storyboard中文本换行</font>__

* __<font size=5>在工程中<font color=red>command + L</font>  :   跳转到具体行</font>__

* __<font size=5><font color=red>按住command + 鼠标左键点击代码</font>  :   实现跳转</font>__

* __<font size=5><font color=red>按住option + 鼠标左键点击代码</font>  :   打开快速帮助</font>__

# <font color=orange>模拟器</font>
* __<font size=5>选中模拟器, <font color=red>command + S</font>  :   给模拟器截图,保存到桌面</font>__

* __<font size=5>选中模拟器, <font color=red>command + shift + H</font>  :   模拟器返回主页</font>__

* __<font size=5>选中模拟器, <font color=red>command + shift + H(点两次)</font>  :   调出模拟器任务栏</font>__

* __<font size=5>选中模拟器, <font color=red>command + shift + K和command + K</font>  :   模拟器输入框键盘开启和取消</font>__

* __<font size=5>选中模拟器, <font color=red>command + 1 或 2 或 3</font>  :   设置模拟器大小</font>__

* __<font size=5>选中模拟器, <font color=red>command + up、down、left、right(方向键)</font>  :   模拟器旋转</font>__

# <font color=orange>Mac快捷键</font>
* __<font size=5>浏览器<font color=red>command + D</font>  :   添加书签</font>__

* __<font size=5>浏览器<font color=red>command +shift + B</font>  :   打开/关闭浏览器上方收藏栏</font>__

* __<font size=5><font color=red>command + 空格</font>  :   打开Spotlight</font>__

# <font color=orange>Mac命令</font>
* __<font size=5>显示隐藏文件：<font color=red>defaults write com.apple.finder AppleShowAllFiles -bool true</font></font>__

* __<font size=5>隐藏隐藏文件：<font color=red>defaults write com.apple.finder AppleShowAllFiles -bool false </font></font>__
上面两个命令执行后需要重启Finder
